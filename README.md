# shutdown-prompt-gui
A simple GUI Prompt (confirmation) to shutdown computer.

This is a very simple AutoHotKey script.

![Screenshot][screenshot]

## License

[MIT License][license]

## Development Dependencies

* [AutoHotKey][ahk]
* [ImageMagick][magick]

## Build

If you have `npm` installed, run:
```
npm run build:icon
npm run build:app
```

Also, you can just run `bat`-files:
```
.\build-icon.bat
.\build-app.bat
```
